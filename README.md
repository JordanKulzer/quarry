# Quarry

A VS Code extension for searching your codebase with multiple 
terms at once.

Instead of running separate searches, add all your terms as chips 
and Quarry finds every file that contains all of them. Each term 
gets a color, and you can click any match to jump straight to that 
line in the editor.

[Install from the VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=jordankulzer.quarry-vscode)

## Packages

- `packages/core` — the search engine, no VS Code dependency
- `packages/vscode-extension` — the VS Code extension
